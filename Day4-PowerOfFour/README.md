<h2>Power of Four</h2>
Given an integer (signed 32 bits), write a function to check whether it is a power of 4.<br><br>

**Example 1:** <br>
Input: 16<br>
Output: true<br>

**Example 2:** <br>
Input: 5<br>
Output: false<br><br>
